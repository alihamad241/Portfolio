This is a comprehensive full-stack e-commerce application for an online bookstore built as a Database course final project. The application features a React + Vite + TailwindCSS frontend and a Node.js + Express backend with MySQL database managed through Sequelize ORM, implementing JWT-based authentication and Stripe payment integration. It provides customers with book browsing by category (Science, Art, Religion, History, Geography), search functionality by title/author/ISBN, shopping cart management, and secure checkout, while administrators can perform full CRUD operations on books, view sales analytics (daily/monthly reports, top customers, best-selling books), and manage publisher orders for inventory replenishment. The database uses a relational model with 9 entities including Users, Books (with ISBN as natural key), Authors, Publishers, Orders, and weak entities for OrderItems, CartItems, and PublisherOrderItems with composite primary keys, featuring intelligent auto-replenishment with threshold-based stock management and historical price snapshots for accurate order records.

Traditional bookstore management systems often struggle with fragmented data handling, where inventory tracking, customer orders, and supplier relationships exist in disconnected silos leading to stock discrepancies, delayed replenishment, and poor sales visibility. Many existing e-commerce solutions lack sophisticated inventory management features such as automatic reordering when stock falls below threshold levels, fail to maintain historical pricing data for accurate financial reporting, and don't provide administrators with real-time analytics on customer behavior and sales trends. Additionally, smaller bookstores need affordable, integrated solutions that combine secure payment processing, user authentication, multi-category book organization, and publisher relationship management without requiring expensive enterprise software or complex manual processes that are prone to human error and inefficiency.

This full-stack online bookstore application solves these challenges by providing a unified, database-driven platform built with React, Node.js, Express, and MySQL that seamlessly integrates all aspects of bookstore operations into a single cohesive system. The solution implements a robust relational database schema with 9 interconnected entities that maintain data integrity through proper foreign key relationships, composite keys for weak entities, and ISBN-based natural keys for books, while Sequelize ORM ensures efficient data operations and migrations. Key features include JWT-secured authentication for customers and administrators, Stripe-integrated payment processing for secure transactions, intelligent threshold-based auto-replenishment that alerts admins when stock levels are low, comprehensive sales analytics dashboards showing daily/monthly trends and top performers, and historical price preservation in order records to maintain accurate financial archives. The system supports full CRUD operations for inventory management, multi-category book organization with advanced search capabilities by title/author/ISBN, real-time shopping cart functionality, and publisher order management, all accessible through an intuitive TailwindCSS-styled interface deployed on Vercel for reliable cloud hosting and scalability.
