This project is a front-end implementation of a modern, fully responsive landing page for Finder, a web platform designed to connect users with top-rated local service professionals. The goal was to create a visually appealing and highly interactive user experience using core web technologies. The page is built with HTML5 and CSS3, leveraging modern layout modules like Flexbox and Grid. All interactive components—including the off-canvas mobile menu, theme switcher, and live theme customizer—are powered by vanilla JavaScript (ES6+), demonstrating a strong command of fundamental web development without reliance on external frameworks. The project emphasizes a clean UI, a seamless responsive design that adapts from mobile to desktop, and advanced features like persistent light/dark mode.

The primary challenge was to build a feature-rich, interactive front-end without relying on a major framework like React or Vue. This required a structured, event-driven approach to manage the state of multiple independent components, including the active theme, the visibility of the off-canvas menu, and the live color values from the theme customizer. A further difficulty was implementing complex, device-specific layouts, such as transforming multi-column grids into horizontally scrolling carousels for mobile devices, which demanded a deep understanding of modern CSS and responsive design principles.

The solution was achieved through a combination of modern CSS architecture and structured, modular JavaScript. The core of the theme and customization system was built using CSS Custom Properties (Variables), allowing JavaScript to easily and efficiently update a single variable and have the change instantly cascade throughout the entire site. All interactivity was handled with a clean, event-driven JavaScript approach where UI states were controlled by toggling specific classes (e.g., offcanvas-open, dark-theme) on the body or html element. To create a seamless and personalized user experience, the user's preferences for the color theme and light/dark mode were saved to the browser's localStorage, ensuring their choices were remembered on their next visit.
